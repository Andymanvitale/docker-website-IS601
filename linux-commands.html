    Linux basic commands

    • pwd: Stands for Print working directory. It gives the absolute path, which means the path that starts from the root.
    • ls: Gives listing in current directory. You can see all the hidden files by using the command "ls -a".
    • cd: Used to navigateion between directories and folders. To go back from a folder to the folder before that, you can type "cd .." . "cd /" takes to root directory.
    • cp: Used to copy files. It takes two arguments: The first is the location of the file to be copied, the second is where to copy.
    • rm: Used to remove or delete files. Use "rm -r" to delete just the directory.
    • mkdir & rmdir: Use mkdir command when you need to create a folder or a directory. Use rmdir to delete an empty directory.
    • man & --help: To know more about a command and how to use it, use the man command "man cd". "cd -help" show which ways the command can be used.
    • locate: Used to locate a file in a Linux system
    • echo: Helps to move some data, usually text into a file. "echo hello, my name is Sowmya >> new.txt"
    • cat: Used to display the contents of a file. It is usually used to easily view programs.
    • nano, vi: nano and vi are already installed text editors in the Linux command line. nano is a good text editor that denotes keywords with color and can recognize most languages. vi is simpler than nano. You can create a new file or modify a file using this editor.
    • df: Used to see the available disk space in each of the partitions in your system.
    • du: Used to know the disk usage of a file in your system.
    • apt-get: Use apt to work with packages in the Linux command line. Use apt-get to install packages.
    • hostname: Use hostname to know your name in your host or network.

    How to log into your docker container.
    Connecting to a running Docker container is helpful when you want to see what is happening inside the container. If the Docker container doesn’t work as expected, you can attach to the container or get a shell to the container and run commands such as "ps" or "top". You can also enter the container, install new packages, and build a new Docker image from it.
    There is a docker exec command that can be used to connect to a container that is already running.

  • Use "docker ps" to get the name of the existing container.
  • Use the command "docker exec -it 'container name' /bin/bash" to get a bash shell in the container
  • Generically, use "docker exec -it 'container name' 'command'" to execute whatever command you specify in the container.
